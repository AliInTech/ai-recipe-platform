# 🍳 AI Recipe Platform (Full Stack SaaS)

An AI-powered full-stack Recipe Platform that allows users to generate, explore, and manage recipes intelligently. This project demonstrates a modern SaaS architecture with authentication, database integration, and AI-driven features.

---

## 🚀 Features

- 🔐 Google Authentication (Secure Login)
- 🤖 AI Recipe Generation based on user input
- 🥫 Pantry-based Recipe Suggestions
- 📚 Headless CMS using Strapi
- 💾 PostgreSQL Database (Neon DB)
- ⚡ Responsive UI with Tailwind CSS & shadcn/ui
- 🛡️ API Security with Arcjet (Rate Limiting & Bot Protection)
- 📊 Subscription & Pricing System

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Next.js (App Router)
- Tailwind CSS
- shadcn/ui

**Backend**
- Strapi (Headless CMS)

**Database**
- PostgreSQL (Neon DB)

**Authentication**
- Google OAuth

**Security**
- Arcjet

---

## 📌 Project Overview

This platform allows users to:
- Generate recipes using AI
- Manage pantry items
- Get personalized recipe suggestions
- Explore recipes by category and cuisine

The application follows a scalable SaaS architecture with secure API handling and modern UI/UX design.
